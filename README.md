# EvilCorp
A lightweight, parody‑themed web sandbox created for experimenting with unusual client‑side behavior, form interactions, and UX patterns. This project is intentionally minimal and exists as a space to test ideas quickly without worrying about structure or long‑term maintenance.

## Overview
EvilCorp is not a production application. It is a personal testing ground where experimental JavaScript behavior, consent popups, form logic, and unconventional UI flows can be explored safely. The name is purely stylistic and does not reflect harmful intent.

## Intent
- Provide a controlled environment for front‑end experiments
- Test how forms, popups, and interaction patterns behave
- Explore edge‑case UX ideas without affecting other projects
- Keep experiments isolated from the main domain

## Status
The project is in its initial stage and will be updated gradually. For now, it serves as a simple foundation for future experiments.

## Disclaimer
Despite the name, this project is strictly for educational and testing purposes. It must not be used to harm users, systems, or services.
